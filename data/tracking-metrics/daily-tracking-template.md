## Decision Framework

### KILL Criteria -- When to Stop/Pause

Pause the campaign and reassess immediately if ANY of the following are true:

| Condition | Threshold | Minimum Data Required | Action |
|-----------|-----------|-----------------------|--------|
| ROAS critically low | ROAS < 0.5 | After EUR 100 spend | **Pause ads immediately.** Review landing page, offer, and targeting. |
| CPA unsustainable | CPA > EUR 75 | After 3+ sales | **Pause ads.** Costs too high to recover even with perfect back-end. |
| Zero conversions | 0 sales | After EUR 150 spend / 100+ clicks | **Pause ads.** Landing page or offer has a fundamental problem. |
| CTR collapsing | CTR < 0.5% | After 1,000+ impressions | **Pause ad group.** Ad copy or targeting mismatch. |
| LP conversion dead | Conv < 1% | After 200+ LP views | **Pause and rebuild.** Landing page failing. |
| CPC too high | CPC > EUR 3.00 | After 50+ clicks | **Pause and adjust bids.** Keywords too competitive or quality score too low. |
| Daily loss limit | Losing > EUR 50/day | After 3 consecutive days | **Reduce budget to EUR 10/day.** Diagnose before scaling back up. |

**Emergency stop:** If total spend exceeds EUR 300 with ROAS below 0.5, kill the campaign entirely and regroup.

### SCALE Criteria -- When to Increase Spend

Increase budget incrementally (20-30% every 3-5 days) when ALL of the following are true:

| Condition | Threshold | Minimum Data Required |
|-----------|-----------|-----------------------|
| ROAS profitable | ROAS > 1.2 | After 10+ sales |
| CPA sustainable | CPA < EUR 40 | Consistent over 5+ days |
| Conversion stable | Conv rate > 3% | Over 300+ LP views |
| CTR healthy | CTR > 1.5% | Over 3,000+ impressions |
| Bump/upsell working | Combined back-end > 30% | After 15+ FE sales |

**Scaling progression:**
1. EUR 20/day (test phase, Days 1-7)
2. EUR 30/day (if KPIs met, Days 8-14)
3. EUR 50/day (confirmed profitability, Days 15-21)
4. EUR 75/day (scaling phase, Days 22-30)
5. EUR 100+/day (proven funnel, Month 2+)

**Rule:** Never increase budget by more than 30% in a single change. Monitor for 3 days after each increase before increasing again.

### PIVOT Criteria -- When to Change Approach

Make specific changes (not full kill) when the following patterns emerge:

| Signal | What It Means | Pivot Action |
|--------|---------------|--------------|
| High CTR + Low conversion (CTR >2%, Conv <2%) | Ads attract clicks but page does not convert | Rewrite landing page. Test new headline, proof, or offer framing. |
| Low CTR + Good conversion (CTR <1%, Conv >4%) | Page works but ads do not attract attention | Rewrite ad copy. Test new headlines, descriptions, extensions. |
| Good FE sales + Low bump rate (<20%) | Bump offer not compelling or poorly positioned | Change bump offer, copy, or price. Test different bump product. |
| Good FE sales + Zero upsells | Upsell page broken or offer mismatch | Check upsell page loads. Test lower price or different product. |
| High CPC + Good conversion | Keywords too expensive but funnel works | Find cheaper keywords. Test display/YouTube. Broaden match types. |
| Good ROAS but low volume | Funnel works but audience too small | Expand keyword list. Test new ad groups. Add display campaigns. |
| Weekday/weekend disparity | Performance varies by day | Adjust ad schedule. Allocate budget to best-performing days. |
| Mobile vs. desktop gap | One device type underperforms | Optimize for winning device. Adjust bids by device. |

**Pivot timeline:**
- Days 1-3: Observe only. Collect data. Make no changes.
- Days 4-7: First optimizations. Pause worst-performing keywords/ads.
- Week 2: Pivot ad copy or landing page if metrics warrant.
- Week 3: Pivot offer structure (pricing, bump, upsell) if back-end underperforms.
- Week 4: Major pivot decision. Continue, restructure, or kill.

## Google Sheets Formulas

Assuming the daily data starts on Row 3, with headers on Row 2. Adjust row numbers as needed.

### Column Layout Reference (for Google Sheets)

| Column | Field |
|--------|-------|
| A | Date |
| B | Ad Spend (EUR) |
| C | Impressions |
| D | Clicks |
| E | CTR (%) |
| F | CPC (EUR) |
| G | LP Views |
| H | Sales (FE) |
| I | Conv Rate (%) |
| J | Rev FE (EUR) |
| K | Bumps Sold |
| L | Bump Rate (%) |
| M | Rev Bumps (EUR) |
| N | Upsells Sold |
| O | Upsell Rate (%) |
| P | Rev Upsells (EUR) |
| Q | Downsells Sold |
| R | Rev Downsells (EUR) |
| S | Total Rev (EUR) |
| T | CPA (EUR) |
| U | ACV (EUR) |
| V | ROAS |
| W | Profit/Loss (EUR) |

### Formulas (Row 3 -- copy down for each day)

```
E3 (CTR %):
=IF(C3>0, (D3/C3)*100, 0)

F3 (CPC):
=IF(D3>0, B3/D3, 0)

I3 (Conversion Rate %):
=IF(G3>0, (H3/G3)*100, 0)

J3 (Revenue Front-end):
=H3*27

L3 (Bump Rate %):
=IF(H3>0, (K3/H3)*100, 0)

M3 (Revenue Bumps):
=K3*17

O3 (Upsell Rate %):
=IF(H3>0, (N3/H3)*100, 0)

P3 (Revenue Upsells):
=N3*67

R3 (Revenue Downsells):
=Q3*27

S3 (Total Revenue):
=J3+M3+P3+R3

T3 (CPA):
=IF(H3>0, B3/H3, 0)

U3 (ACV):
=IF(H3>0, S3/H3, 0)

V3 (ROAS):
=IF(B3>0, S3/B3, 0)

W3 (Profit/Loss):
=S3-B3
```

### Weekly Summary Formulas (assuming daily data in rows 3-9 for Week 1)

```
Total Spend:        =SUM(B3:B9)
Total Impressions:  =SUM(C3:C9)
Total Clicks:       =SUM(D3:D9)
Avg CTR:            =IF(SUM(C3:C9)>0, (SUM(D3:D9)/SUM(C3:C9))*100, 0)
Avg CPC:            =IF(SUM(D3:D9)>0, SUM(B3:B9)/SUM(D3:D9), 0)
Total LP Views:     =SUM(G3:G9)
Total Sales:        =SUM(H3:H9)
Avg Conv Rate:      =IF(SUM(G3:G9)>0, (SUM(H3:H9)/SUM(G3:G9))*100, 0)
Total Rev FE:       =SUM(J3:J9)
Total Bumps:        =SUM(K3:K9)
Bump Rate:          =IF(SUM(H3:H9)>0, (SUM(K3:K9)/SUM(H3:H9))*100, 0)
Rev Bumps:          =SUM(M3:M9)
Total Upsells:      =SUM(N3:N9)
Upsell Rate:        =IF(SUM(H3:H9)>0, (SUM(N3:N9)/SUM(H3:H9))*100, 0)
Rev Upsells:        =SUM(P3:P9)
Total Downsells:    =SUM(Q3:Q9)
Rev Downsells:      =SUM(R3:R9)
Total Revenue:      =SUM(S3:S9)
Avg CPA:            =IF(SUM(H3:H9)>0, SUM(B3:B9)/SUM(H3:H9), 0)
Avg ACV:            =IF(SUM(H3:H9)>0, SUM(S3:S9)/SUM(H3:H9), 0)
ROAS:               =IF(SUM(B3:B9)>0, SUM(S3:S9)/SUM(B3:B9), 0)
Profit/Loss:        =SUM(S3:S9)-SUM(B3:B9)
```

### Conditional Formatting Rules (apply to full data range)

```
CTR (Column E):
  Green: >= 2.0
  Yellow: >= 1.5 AND < 2.0
  Red: < 1.5

CPC (Column F):
  Green: <= 1.50
  Yellow: > 1.50 AND <= 2.00
  Red: > 2.00

Conversion Rate (Column I):
  Green: >= 4.0
  Yellow: >= 3.0 AND < 4.0
  Red: < 3.0

Bump Rate (Column L):
  Green: >= 40
  Yellow: >= 30 AND < 40
  Red: < 30

CPA (Column T):
  Green: <= 35
  Yellow: > 35 AND <= 50
  Red: > 50

ROAS (Column V):
  Green: >= 1.5
  Yellow: >= 1.0 AND < 1.5
  Red: < 1.0

Profit/Loss (Column W):
  Green: > 0
  Red: <= 0
```
